The IMDB Sentiment Analysis.ipynb contains the steps necessary to reproduce the results in our report and for the leader board model.
The sentiment+naivebayes.py file contains the Bernoulli Naive Bayes implementation.

Dependencies:
os
re
numpy
pandas
sklearn: model_selection, feature_extraction, preprocessing, svm, pipeline, metrics, tree, linear_model
json
matplotlib.pyplot
itertools
textblob
time
